# Development
## Prerequisites
Have [ffmpeg](https://formulae.brew.sh/formula/ffmpeg#default) installed

## Audio
To compress any audio file down to 400kb use the Makefile in /Audio folder by running:

```
make input="yourAudioFile.mp3"   
```
